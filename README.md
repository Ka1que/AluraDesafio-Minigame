# Pequenas anotações Alura Curso

São apenas algumas anotações e a classe AkINTor é um minigame que foi sugerido como desafio 

ele consiste em um numero gerado de forma aleatória de 0 a 100 e o usuario vai precisar adivinhar

são apenas 5 tentativas e o programa vai dando dica se é maior ou menor que a ultima tentativa

## 

obs. O jogo é pelo console e para jogar novamente é necessário executar novamente o programa
